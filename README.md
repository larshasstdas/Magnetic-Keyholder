# Modular Hexagonal Magnetic Key Holder

A modular, wall-mounted key holder system built around a hexagonal tile design. Tiles can be connectec via the connector piece and can be freely arranged and rearranged on the wall (wall need to be metallic/magnetic).

---

## Features

- **Modular design** — tiles connect to each other and can be extended in any direction
- **Hexagonal geometry** — compact and visually clean tiling pattern
- **Magnetic mounting** — no tools or screws required to reconfigure the layout

---

## Hardware

Neodymium magnet with 12mm in diameter are glued into the cavitys at the bottom on each part. The hexagons holding keys require 2 magnets and the plain hexagons 1.

---

## Printing

STL files are ready to slice — no CAD software needed.


---

## Files

```
creofiles                     — all files used for creating the stl-files and assembly model
keyholder_hook.stl            — main tile for holding the key
keyholder_plain.stl           — blank decorative tile
keyholder_connection.stl      — part for connecting tiles
```

---

## Assembly

1. Glue neodymium magnets (12 mm diameter) into the cavitys on each tile.
2. Connect tiles with the connecting piece.
3. Attach to the wall.

---

## CAD Source

The project was designed in **Creo Parametric 12**. Source files are in the creofiles folder in this repository.
